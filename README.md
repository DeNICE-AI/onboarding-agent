# Onboarding AI Assistant

ИИ-ассистент для онбординга новых сотрудников. Проект создан для автоматизации процесса адаптации: ответов на частые вопросы, предоставления чеклистов и снижения нагрузки на HR-отдел.

## Описание проекта

Нейроассистент помогает новым сотрудникам адаптироваться в компании:
- Отвечает на вопросы о документах, графике и рабочих инструментах.
- Предоставляет пошаговые чек-листы на испытательный срок.
- Направляет сложные запросы к профильным специалистам или в службу технической поддержки (support@company.com).
- Сохраняет историю контекста текущей беседы (память на последние 20 сообщений) для более естественного диалога.

## Системный промпт
```text
Ты ИИ-ассистент для онбординга новых сотрудников компании. Твоя цель — помочь новичкам адаптироваться: отвечать на частые вопросы о документах, графике, рабочих инструментах, предоставлять чеклисты. 
Стиль общения: доброжелательный, структурированный, корпоративный. 
Ограничения: ты не принимаешь кадровые решения и не даёшь юридических консультаций. Если в предоставленном контексте нет ответа на вопрос, КАТЕГОРИЧЕСКИ ЗАПРЕЩАЕТСЯ придумывать информацию. В этом случае честно ответь, что не знаешь, и предоставь контакты технической поддержки: email support@company.com, телефон +7 (800) 123-45-67. 
Используй только предоставленный контекст для формирования ответа.
```

## Сценарии и примеры диалогов

**Сценарий 1: Вопрос по доступу**
- *Пользователь*: Как мне зайти в корпоративную почту?
- *Ассистент*: Добро пожаловать в команду! Доступ к корпоративной почте предоставляется в первый рабочий день. Пароль был выслан вам по SMS. Для входа перейдите на mail.company.com. Если SMS не пришло, пожалуйста, обратитесь в IT-отдел (внутренний номер 112).

**Сценарий 2: Вопрос с эскалацией**
- *Пользователь*: Могу ли я взять отпуск авансом?
- *Ассистент*: К сожалению, я не могу принимать кадровые решения. По вопросам оформления отпусков, пожалуйста, обратитесь к HR-специалисту вашего департамента. 

## База знаний и актуализация
База знаний хранится в файле `data/knowledge_base.csv`.

**Инструкция по актуализации:**
1. Откройте файл `data/knowledge_base.csv` в любом табличном редакторе или текстовом редакторе.
2. Файл состоит из двух колонок: `question` (тема/вопрос) и `answer` (инструкция/ответ).
3. Добавьте новую строку с вопросом и подробным ответом.
4. После сохранения файла необходимо пересобрать индекс RAG:
   ```bash
   python -m backend.build_index
   ```

## Запуск
1. Установите зависимости: `pip install -r requirements.txt`.
2. Настройте файл `.env` с ключами `GIGACHAT_CLIENT_ID` и `GIGACHAT_CLIENT_SECRET`.
3. Соберите индекс: `python -m backend.build_index`.
4. Запустите сервер: `uvicorn backend.app:app --host 0.0.0.0 --port 8001`.
5. Откройте `frontend/index.html` в браузере.
