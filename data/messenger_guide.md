# Использование корпоративного мессенджера

Для оперативной связи внутри ГК ТОФС используется корпоративный мессенджер eXpress.

## Как получить доступ
Приложение eXpress установлено на всех корпоративных ПК по умолчанию.
Для входа введите ваш корпоративный email и пароль от Active Directory.

## Мобильная версия
1. Скачайте приложение eXpress из App Store или Google Play.
2. При первом запуске выберите "Корпоративный сервер".
3. Введите адрес сервера: `chat.tofs.ru`.
4. Авторизуйтесь с помощью корпоративной учетной записи.

*Использование публичных мессенджеров (Telegram, WhatsApp) для передачи корпоративной информации строго запрещено политикой ИБ.*
