# Политика информационной безопасности

Ключевые правила ИБ, которые обязан соблюдать каждый сотрудник ГК ТОФС.

## Основные положения
1. **Блокировка экрана:** Обязательно блокируйте экран ПК (`Win + L`) при уходе с рабочего места.
2. **Пароли:** Никому не передавайте свой пароль. Техподдержка SOLV никогда не попросит вас назвать пароль.
3. **Фишинг:** Будьте внимательны к письмам от неизвестных отправителей. Если письмо кажется подозрительным, перешлите его на `security@tofs.ru` с пометкой "Спам".
4. **Съемные носители:** Использование личных флеш-накопителей на корпоративных ПК запрещено. Порты USB заблокированы для записи.
