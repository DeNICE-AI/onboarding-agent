# Правила удаленной работы

Для сотрудников, которым согласован удаленный или гибридный формат работы, действуют следующие ИТ-правила.

## Подключение из дома
Все подключения должны осуществляться только через корпоративный VPN-клиент. 

## Использование личного ПК (BYOD)
Доступ к корпоративным ресурсам с личных компьютеров разрешен только через инфраструктуру VDI (Virtual Desktop Infrastructure).
1. Установите клиент VMware Horizon на свой личный ПК.
2. Подключитесь к серверу `vdi.tofs.ru`.
3. Работайте в безопасной виртуальной среде. Прямое копирование файлов с VDI на личный ПК заблокировано.
