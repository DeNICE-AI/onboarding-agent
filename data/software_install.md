# Установка разрешенного ПО

Для установки программного обеспечения в ГК ТОФС используется центр приложений.

## Инструкция
1. Откройте "Центр программного обеспечения" (Software Center) из меню Пуск.
2. В разделе "Приложения" выберите нужную программу (например, 7-Zip, Adobe Reader, Google Chrome).
3. Нажмите "Установить".
4. Дождитесь завершения установки (может потребоваться перезагрузка).

## Установка нестандартного ПО
Если нужной программы нет в Центре ПО, вам необходимо создать заявку в систему SOLV. Укажите название программы, версию и бизнес-обоснование необходимости установки. Установка будет произведена после согласования с вашим руководителем и отделом информационной безопасности.
